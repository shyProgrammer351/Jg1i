11. 클래스/패키지

접근권한 class 클래스이름 {
  접근권한 자료형 변수이름;
  접근권한 final 자료형 상수이름;
  ...
 
 접근권한 클래스이름( 자료형 x1, ... , 자료형 xn ) {
        초기화 내용
    } 		※ 객체 = new 클래스이름( x1, ..., xn );
   생성자 : x를 바탕으로 객체 초기화
 

 접근권한 자료형y 메서드이름( 자료형 x1, ... , 자료형 xn ) {      
        실행 내용
        return y;
    }
※ y = f( x1, x2, ..., xn )
   메서드 f : x를 바탕으로 실행 후 y 반환
   입력 x : 메서드에 전달하는 값
   출력 y : 메서드의 실행 결과값
  ...
}

class   객체의 구조를 선언한 설계도로서 서로 관련이 있는 변수, 상수, 메서드를 포함

객체(object), 인스턴스(instance)   클래스 구조에 따라 메모리 공간이 확보되어 실질적으로 생성된 상태
예) 휴대폰 클래스 - 내 휴대폰 객체, OO친구 휴대폰 객체, XX친구 휴대폰 객체
※ 휴대폰 구조나 기능은 동일하더라도, 각 객체에 저장된 연락처, 문자메시지 등 저장된 데이터는 다름

Object   모든 클래스의 부모가 되는 최상위 클래스로 equals(), toString() 메서드를 포함

생성자   클래스 객체 상태를 전달인자를 바탕으로 초기화하며, 반환값이 없고 클래스이름과 동일한 메서드

method   y = f( x1, x2, ..., xn )처럼 0개 이상의 x값을 전달받아 실행하고 그 결과값을 y로 반환

return   메서드 종료시 실행흐름이 메서드를 호출했던 곳으로 되돌아가는 명령문

new   클래스 객체 생성를 위해 메모리 공간을 확보하고, 클래스 객체 상태를 초기화하기를 위해 생성자 호출

final   변수 공간에 변하지 않는 상수값을 저장한다고 선언하는 키워드

접근권한   해당요소에 접근할 수 있는 범위를 지정하며, 접근제어자 public, protected, private으로 구분

public   모든 클래스에서 접근할 수 있도록 허용

protected   자식 클래스나 동일 패키지내 다른 클래스에서만 접근할 수 있도록 허용

private   해당 클래스 내에만 접근할 수 있도록 허용

접근제어자생략(default)   접근제어자가 없으면, 동일 패키지내 다른 클래스에서만 접근할 수 있도록 허용

클래스 예제 : 점 (x,y)

import java.lang.Math;  
public class XYpoint { 클래스 시작
    private int x, y; 변수 선언
    public XYpoint( int x, int y ) {
        this.x = x;
        this.y = y;
    } 		생성자1
    public XYpoint() {
        this( 0, 0 );
    } 		생성자2
    public String toString() {
        return "(" + this.x + "," + this.y + ")";
    } 		toString 메서드
    public boolean equals( XYpoint that ) {
        return ( this.x == that.x) && ( this.y == that.y );
    } 		equals 메서드
    public double compareTo( XYpoint that ) {
        return Math.sqrt( Math.pow( ( this.x - that.x ), 2 ) + Math.pow( ( this.y - that.y ), 2 ) );
    } 		compareTo 메서드
} 클래스 끝
import java.util.Scanner;  
public class Main { 클래스 시작
    public static void main(String[] args) {
        Scanner scan = new Scanner( System.in );
        System.out.print( "점의 x좌표 값을 입력하세요 : " );
        int x = scan.nextInt();
        System.out.print( "점의 y좌표 값을 입력하세요 : " );
        int y = scan.nextInt();

        XYpoint origin = new XYpoint();
        XYpoint point = new XYpoint( x, y );

        if ( origin.equals( point ) ) {
          System.out.println( "이 점은 원점입니다" );
        }
        else {
          System.out.println(origin+"과"+point+"사이의 거리는"+point.compareTo(origin)+"입니다");
        }
    } 		프로그램 시작

this   객체 자신에 대한 레퍼런스

toString   해당 객체를 대표하는 문자열로 변환하는 메서드

equals   해당 객체와 전달받은 객체의 대표값이 같은지 판정하는 메서드

import   다른 패키지 클래스를 가져오기

Math   수학과 관련된 다양한 메서드를 제공해주는 클래스

sqrt   square root의 약자로 전달인자의 제곱근 결과를 반환하는 메서드

pow   power의 약자로 pow(밑수,지수)는 밑수의 지수승을 반환하는 메서드. 예를 들어, pow(3,4) = 34 = 81

변수의 유효 범위

클래스 변수   클래스에서 static으로 선언한 변수이며, 클래스로 생성된 모든 객체 인스턴스에서 공유 가능

인스턴스 변수   객체 인스턴스를 생성할 때마다 변수 공간을 확보하고, 클래스내 모든 메서드에서 공유 가능

지역변수   변수를 선언한 메서드에서만 접근가능하며, 메서드 호출시 변수 공간을 생성하고 메서드 종료시 소멸

public class Main {
    public static void main(String[] args) {
        int localVariable = 0;
   
        VariableScope obj1 = new VariableScope();
        obj1.add1();
        localVariable += 100;
        obj1.add2();
        localVariable += 100;
   
        VariableScope obj2 = new VariableScope();
        obj2.add2();
        localVariable += 100;
    }
}


  		class VariableScope {
    private static int classVariable = 0;
    private int instanceVariable = 0;
   
    public void add1() {
        int localVariable = 0;
        localVariable++;
        instanceVariable++;
        classVariable++;
    }
   
    public void add2() {
        int localVariable = 0;
        localVariable += 10;
        instanceVariable += 10;
        classVariable += 10;
    }
}
