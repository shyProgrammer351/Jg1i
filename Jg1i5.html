문제

국민은행, 우리은행, 하나은행에서 통장을 만들어 각각 1000원, 2000원, 3000원을 입금 후 총예금액을 출력하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
국민은행 : 1000원  총예금액 : 1000원
우리은행 : 2000원  총예금액 : 3000원
하나은행 : 3000원  총예금액 : 6000원
(전체 총예금액) 6000원

알고리즘

 통장 클래스 
    통장 개설 
    통장 입금 
    통장 내역을 문자열로 표현 
    전체 통장의 총예금액을 반환 
 
 프로그램 시작 
    국민은행 통장 만들어 1000원 입금 후 출력 
    우리은행 통장 만들어 2000원 입금 후 출력 
    하나은행 통장 만들어 3000원 입금 후 출력 
    전체 전체 총예금액 출력 
 프로그램 종료
